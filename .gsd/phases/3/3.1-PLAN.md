# Plan 3.1: Serial Communication Service

<objective>
To create a central service layer for managing USB OTG Serial connections. Because Expo Go and Web don't natively support raw Android USB OTG out of the box without a custom dev client, this service should use a Mock Mode to simulate connections, enabling rapid UI prototyping for the rest of the application.
Purpose: Decouple the hardware logic from the React UI components.
Output: A Singleton or Provider pattern for `SerialService`.
</objective>
